All features are pre-match data only:

| Feature | Description |

| `over_under_cap` | The line/cap (e.g., 2.5, 2.75, 3.0) |
| `implied_over` | Implied probability of Over (1/over_odd) |
| `implied_under` | Implied probability of Under (1/under_odd) |
| `odds_ratio` | over_odd / under_odd |
| `cap_2_5_2_75` | Binary: cap in 2.5-2.75 range |
| `cap_2_75_3_0` | Binary: cap in 2.75-3.0 range |
| `cap_3_0_3_5` | Binary: cap in 3.0-3.5 range |
| `cap_high` | Binary: cap > 3.5 |
| `cap_low` | Binary: cap <= 2.5 |
| `cap_norm` | Normalized cap value |
| `home_odd` | 1X2 home win odds |
| `away_odd` | 1X2 away win odds |
| `draw_odd` | 1X2 draw odds |
| `home_prob` | Implied home win probability |
| `away_prob` | Implied away win probability |
| `favor_over` | Binary: market favors Over |
| `market_even` | Binary: market is even |
| `home_cap` | Asian handicap home cap |
| `home_cap_abs` | Absolute value of handicap cap |
